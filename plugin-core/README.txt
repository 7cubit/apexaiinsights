Apex AI Insights brings real-time analytics, AI-powered chat (`QueryBuilder`), and True Profit tracking to your WooCommerce store.

== Description ==

Turn your data into decisions with Apex AI Insights. This plugin connects your store to a powerful external Go Engine that processes millions of events in real-time.

**Key Features:**
*   **Live Dashboard**: Real-time traffic, sales, and user activity.
*   **Intelligence Engine**: Ask "Why is traffic down?" and get an answer backed by SQL data.
*   **True Profit**: Track Net Profit by deducting COGS and Gateway Fees.
*   **Whale Alerts**: Get notified when a cart exceeds your custom threshold.
*   **Inventory Prediction**: Know exactly when you'll run out of stock.

== Installation ==

1. Upload the `apex-ai-insights.zip` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to `Apex AI Insights` > `Settings` to configure your License Key and API keys.
4. Ensure the Apex Go Engine is running (Docker required for the engine).

== Frequently Asked Questions ==

= Does this require an external service? =
Yes, it connects to the Apex Go Engine (included in the Docker setup).

= Is it compatible with WooCommerce? =
Yes, it includes deep integration for Orders and Product Data.
